# Change Color Box 🎨

This is a simple interactive color box application built using HTML, CSS, and JavaScript.  
Users can change the background color of the box using preset buttons (Yellow, Blue, Red) or generate a random hex color.

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## ⚙️ Features
- Changes color based on button selection
- Random color generator using JavaScript
- Clean layout and responsive buttons

## 💡 Example Colors
- `Шар` → `#FFFF00`
- `Цэнхэр` → `#0000FF`
- `Улаан` → `#FF0000`
- `Санамсаргүй` → randomly generated `#xxxxxx` hex

## 🧪 How to Use
1. Clone or download this repo
2. Open `index.html` (or `color-box.html`) in your browser
3. Click any of the buttons to see color changes!
